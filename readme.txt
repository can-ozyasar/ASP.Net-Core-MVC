btkakademi ASP.NET CORE MVC EĞİTİM serisi içerisindeyken aldığım notlar kodları içeren repodur.


dotnet new mvc -o Basics   il basics isminde bir mvc projesi oluşturulur 

dotnet build ile proje derlenir

dotnet run ile proje çalıştırılır .




default değerlerde localhost:5163  localhost:5163/Home   localhost:5163/Home/Index   bu üç adress de  aynı sayfayı açar yani ilk karşımıza gelen sayfa 



dotnet watch ile her değişiklik güncelleme sonucunda run etmemize gerek olmaz .o otomatik olarak güncellemeleri ekrana yansıtır .

dotnet new gitignore   komutu ile gereksiz dosyaların githuba gitmesini engelleriz .
